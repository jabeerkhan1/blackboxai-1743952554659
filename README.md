# Palmpay Clone

## Project Overview
Palmpay Clone is a web-based application that simulates the functionalities of the Palmpay platform. It includes a login interface and a dashboard for user interactions, making it a great starting point for learning about web development with modern technologies like Tailwind CSS and Font Awesome.

## Installation
To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/palmpay-clone.git
   ```

2. **Navigate into the project directory**:
   ```bash
   cd palmpay-clone
   ```

3. **Open the `index.html` or `dashboard.html` file in your browser** to view the application:
   - You can double-click on the HTML files or open them using any web server.

## Usage
- **Login**: Users can enter their email and password to log into the platform. Options for remembering login details and recovering passwords are included.
- **Dashboard**: After logging in, users will be redirected to the dashboard where they can view their balance, recent transactions, and quick action buttons to send or request money.

## Features
- **Responsive Design**: Built with Tailwind CSS for a modern and responsive user interface.
- **Login Form**: Includes fields for email, password, and a "Remember Me" checkbox.
- **User Interaction**: Featuring buttons for sending money, requesting money, and viewing transaction history on the dashboard.
- **Recent Transactions Display**: Shows recent activity for better transaction management.

## Dependencies
This project leverages the following dependencies:
- **Tailwind CSS**: For styling the application.
- **Font Awesome**: For including icons in the user interface.

External stylesheets and scripts are linked directly from CDN, so there is no need for additional installation.

## Project Structure
Here's a brief overview of the project structure:

```
palmpay-clone/
│
├── index.html         # Login page
├── dashboard.html     # User dashboard page
```

### index.html
- Contains the login page interface, including form inputs and links for user interaction.

### dashboard.html
- Displays the user dashboard with balance information and actions like sending money and viewing transaction history.
